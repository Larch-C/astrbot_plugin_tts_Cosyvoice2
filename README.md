# astrbot_plugin_tts_Cosyvoice2

Astrbot的tts功能补充（使用Cosyvoice2-0.5b本地模型）

# 使用

先将Cosyvoice克隆至Astrbot的主目录下(eg. /Astrbot)

再按[Cosyvoice官方文档](https://www.modelscope.cn/models/iic/CosyVoice2-0.5B/summary)下载Cosyvoice0.5b模型

配置好环境后便可使用

eg. /tts 传入llm的内容
